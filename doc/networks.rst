#############################
Networks
#############################
.. _ppNetworks:


Besides creating your own grids through the pandapower API pandapower provides synthetic and Benchmark networks through the networks module.
The pandapower networks modul contains simple test networks, randomly generated networks, CIGRE test networks, IEEE case files and synthetic networks from the dissertation of Georg Kerber and Lindner et. al..
